# Zomato Clone 🍔

A Zomato clone web application built as part of the Edunet program.

## Live Demo
🔗 https://zomoto-clone-weld.vercel.app/

## Features
- Restaurant listing
- Food search
- Responsive UI
- Modern frontend design

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB

## Installation
1. Clone the repository
2. Run npm install
3. Run npm start
